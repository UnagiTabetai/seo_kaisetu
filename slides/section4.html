競合分析AIプロンプト

「[ターゲットキーワード]」で上位表示されている
競合サイトの分析を行い、キーワード戦略を提案してください：

【分析項目】
1. 上位10サイトの共通要素
   - 扱っているトピック
   - 見出し構造の特徴
   - 平均文字数
   - 使用キーワード

2. 差別化ポイントの発見
   - 競合が扱っていない情報
   - より深く掘り下げられる領域
   - 独自の視点や切り口

3. コンテンツギャップ
   - ユーザーニーズで未対応の部分
   - より具体的なロングテールキーワード

4. 改善戦略の提案
   - 競合を上回るコンテンツ構成
   - 追加すべき独自要素
   - E-E-A-T強化のポイント

2024年最新のSEOトレンド

AI生成コンテンツ

Googleの対応：

  • 品質重視（生成方法は問わない）
  • E-E-A-Tがより重要に
  • 人間の編集・検証が必須
ベストプラクティス： AIで下書き → 人間が編集・事実確認 → 独自の洞察追加

音声検索最適化

対策ポイント：

  • 会話調の自然な文章
  • 質問形式のキーワード
  • ローカルSEOの強化
キーワード例： 「近くの〇〇はどこ」「〇〇のやり方を教えて」

Helpful Content Update

重視される要素：

  • ユーザーファーストの内容
  • 独自の価値・経験
  • 検索意図の完全な満足
避けるべき： SEOのためだけの薄いコンテンツ

Core Web Vitals

重要指標：

  • LCP（読み込み速度）< 2.5秒
  • FID（応答性）< 100ms
  • CLS（視覚的安定性）< 0.1
影響： ユーザー体験とランキングに直結

今後の展望

AIとの共存：SGE（Search Generative Experience）への対応準備

動画コンテンツ：YouTube動画のSEO重要性増大
